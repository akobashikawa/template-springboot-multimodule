# Template Simple - Spring Boot 3.5.x

Este es un módulo base autónomo diseñado para ser clonado e independizado.

## 🚀 Cómo usar este template
1. Copia la carpeta `template-simple` a tu nuevo repositorio.
2. Cambia el `<artifactId>` y `<groupId>` en el `pom.xml`.
3. Actualiza el `spring.application.name` en `application.yml`.

## 🛠 Especificaciones Técnicas
- **Java:** 21
- **Framework:** Spring Boot 3.5.11
- **Lombok:** Habilitado con Annotation Processing.
- **Versionado:** CalVer (YYMMDD-SemVer).

## 💻 Desarrollo Local
Para ejecutar con el perfil local:
`./mvnw spring-boot:run -Dspring-boot.run.profiles=local`
